# cc-switchy v0.4.1

## English

### Fixes & Improvements

1. Additive Agents now show the number of enabled providers directly in the header.
2. Provider rows now use clear enabled and disabled states, while Agents without providers show "No providers" directly.

## 简体中文

### 修复与改进

1. 累加式智能体的顶部状态现在直接显示已启用的供应商数量。
2. 供应商列表现在明确显示“已启用”或“未启用”，没有供应商时直接显示“无供应商”。
